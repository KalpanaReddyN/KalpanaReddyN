<h3 align="center">DevOps and Cloud Engineer</h3>

<p>I'm a DevOps Engineer with a strong passion for automation, cloud infrastructure, and continuous integration & delivery (CI/CD). I thrive on designing and optimizing scalable, secure, and highly available systems that improve software delivery and operational efficiency.

I specialize in Cloud Platforms like AWS and leverage tools like Terraform, Ansible, and Helm to implement Infrastructure as Code (IaC) for better automation and consistency. My experience includes containerization and orchestration using Docker and Kubernetes, enabling seamless deployments across environments. I also work with CI/CD pipelines using Jenkins, GitHub, and Helm charts, ensuring smooth software releases with minimal downtime.

Beyond automation, I focus on monitoring and observability, integrating tools like Prometheus, Grafana, and ELK Stack to track system performance and proactively address issues. I'm always exploring new DevOps technologies, best practices, and security enhancements to make infrastructure more resilient, efficient, and cost-effective.

I'm open to collaborating on Cloud Infrastructure Projects, automation workflows, and DevOps best practices. Whether it’s optimizing CI/CD pipelines, implementing Kubernetes-based architectures, or automating cloud deployments, I’m always excited to learn, innovate, and contribute.

Always exploring new DevOps & Cloud technologies and eager to collaborate on exciting projects!
Let’s connect and build something amazing together!</p>
